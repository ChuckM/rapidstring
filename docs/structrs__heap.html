Detailed Description

A struct that stores the heap data and a flag with explicit alignment. Accessing packed data structures incurs a performance penalty, therefore the alignment will be used to allow for a larger stack string.

Field Documentation

◆ buffer

char* rs_heap::buffer

The buffer of a heap string allocated using RS_ALLOC or RS_REALLOC. This buffer may be manually freed by directly calling RS_FREE(s->buffer, s->capacity + 1). Doing so will avoid a flag check. The additional one is for the null terminator, which is subtracted upon initial allocation.

◆ flag

uint8_t rs_heap::flag

The following flag is used to store the state of the union as well as the remaining capacity of the stack string. They may be shared as the capacity of a stack string will always be smaller than RS_HEAP_FLAG, which guarentees unambiguity between the two states.
