Detailed Description

Struct that stores the heap data.

Accessing packed data structures incurs a performance penalty, therefore the alignment will be used to allow for a larger stack string.

Field Documentation

◆ buffer

char* rs_heap::buffer

Buffer of a heap string.

Allocated using RS_MALLOC or RS_REALLOC. This buffer may be manually freed by directly calling RS_FREE(s->buffer). Doing so will avoid the heap flag check. The additional one is for the null terminator, which is subtracted upon initial allocation.

unsigned char rs_heap::flag

Flag of the rapidstring union.

The following is used to store the state of the union as well as the remaining capacity of the stack string. They may be shared as the capacity of a stack string will always be smaller than #RS_HEAP_FLAG, which guarentees unambiguity between the two states.
